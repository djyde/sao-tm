# sao-tm

> create new typescript module

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao tm my-project
```

### From git

```bash
sao djyde/sao-tm my-project
```
